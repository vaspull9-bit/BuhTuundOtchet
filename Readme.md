BuhTuundOtchet

BuhTuundOtchet — это современная настольная программа для бухгалтеров, работающих в 1С. Она позволяет загружать выгрузки стандартных отчётов из 1С (в формате Excel), автоматически рассчитывать ключевые показатели (выручка, себестоимость, валовая и чистая прибыль, НДС продажи) и формировать наглядные отчёты с графиками.

Программа разработана специально для нужд российского бухгалтерского учёта и поддерживает работу с двумя компаниями одновременно, детализацию по товарам и товарным группам, а также экспорт результатов в Excel, PDF и Word.

🚀 Возможности
Импорт данных из Excel‑файлов, выгруженных из 1С (поддерживаются русские названия колонок).

Автоматический расчёт валовой и чистой прибыли, НДС продажи в бюджет.

Хранение истории загрузок во встроенной базе данных SQLite.

Удобная фильтрация по компаниям, периодам и товарным группам.

Интерактивная таблица с сортировкой по любому столбцу и цветовым выделением сумм НДС.

Визуализация данных (4 вида графиков) на отдельной вкладке.

Экспорт отчётов в форматы:

Excel — с автоформатированием и итогами;

PDF — с поддержкой кириллицы, графиками и таблицами;

Word — с встроенными диаграммами.

Быстрый отчёт по кнопке на панели инструментов.

Современный интерфейс в тёмных тонах, удобный для ежедневной работы.

🖥️ Системные требования
Операционная система: Windows 7/10/11 (64‑bit)

Установленный Python 3.8 или выше (если вы запускаете из исходного кода)

Не менее 2 ГБ оперативной памяти

Свободное место: около 100 МБ (плюс место для сохраняемых отчётов)

📁 Подготовка данных в 1С
Программа работает с Excel‑файлами, выгруженными из стандартных отчётов 1С. Для корректной работы необходимо подготовить сводный файл, содержащий следующие колонки (названия могут быть как на русском, так и на английском — программа поймёт оба варианта):

Русское название	Английское название	Обязательное	Описание
Период	period	✅	Месяц в формате ММ.ГГГГ
Компания	company	✅	Наименование организации
Товарная группа	product_group	✅	Группа номенклатуры
Номенклатура	nomenclature	✅	Наименование товара/услуги
Выручка (с НДС)	revenue	✅	Сумма продаж с НДС
НДС в выручке	vat_sales	✅	НДС, начисленный с продаж
Себестоимость	cost_price	✅	Себестоимость реализованных товаров
НДС покупки	vat_deductible	❌	Входящий НДС (из книги покупок)
Расходы на продажу	sales_expenses	❌	Коммерческие расходы (счёт 44)
Прочие доходы/расходы	other_income_expenses	❌	Сальдо прочих доходов/расходов
Оборот (кол-во)	quantity	✅	Количество проданных единиц
💡 Если каких‑то колонок нет (например, расходы на продажу или НДС покупки), программа автоматически подставит нули и рассчитает недостающие показатели на основе имеющихся данных. Но для точного расчёта НДС продажи необходимо наличие колонок НДС в выручке и НДС покупки.

Как получить данные из 1С?
Подробная пошаговая инструкция по формированию каждого отчёта находится в файле INSTRUCTION.md (или см. раздел «Инструкция для пользователя» в документации). Вкратце, вам потребуются:

ОСВ по счёту 90 (с детализацией по номенклатуре)

ОСВ по счёту 91

ОСВ по счёту 41

Анализ счёта 44

Отчёт по продажам (валовая прибыль)

Книга покупок и книга продаж (для НДС)

После выгрузки каждого отчёта в Excel вы можете объединить нужные данные в единый файл по приведённому выше шаблону или загружать их по отдельности — программа умеет добавлять записи из нескольких файлов.

🖥️ Интерфейс и основные функции
Главное окно
Панель инструментов (тулбар): кнопки импорта, экспорта, быстрого отчёта и информации о программе.

Панель фильтров: выбор компании, периода и товарной группы.

Вкладки:

«Таблица данных» — здесь отображаются все загруженные записи с возможностью сортировки и подсветкой сумм НДС.

«Графики и анализ» — четыре автоматически обновляемых графика (распределение прибыли, НДС по компаниям, динамика выручки, ТОП‑5 товаров).

Строка итогов под таблицей показывает общую выручку, НДС продажи и чистую прибыль по текущему фильтру.

Импорт данных
Нажмите кнопку «Загрузить Excel» (иконка папки).

Выберите один или несколько файлов (формат .xlsx или .xls).

Программа проверит наличие обязательных колонок, переименует русские заголовки в английские и сохранит данные в базу.

После успешной загрузки таблица и графики обновятся автоматически.

Фильтрация и сортировка
Используйте выпадающие списки сверху для выбора компании, периода или товарной группы.

Нажмите «Применить фильтр» — таблица и графики перестроятся.

Чтобы отсортировать данные по любому столбцу, кликните по заголовку столбца.

Экспорт отчётов
На панели инструментов доступны три кнопки экспорта:

📊 Экспорт в Excel — сохраняет текущий набор данных (с учётом фильтра) в файл .xlsx с автоформатированием и отдельным листом «Итоги».

📄 Экспорт в PDF — создаёт профессиональный отчёт с графиками, таблицей и итоговыми показателями. Кириллица поддерживается (используется шрифт Arial).

📝 Экспорт в Word — формирует документ .docx с заголовками, таблицей и встроенным изображением графиков.

Быстрый отчёт
Кнопка «📋 Быстрый отчёт» показывает всплывающее окно с ключевыми цифрами и ТОП‑5 товаров по прибыльности — удобно для оперативной сверки.

🧮 Расчёт НДС продажи
Главная задача программы — помочь бухгалтеру быстро получить сумму НДС, подлежащую уплате в бюджет, которая рассчитывается как:

text
НДС продажи = НДС в выручке (книга продаж) – НДС к вычету (книга покупок)
Программа автоматически суммирует эти показатели по всем записям с учётом текущих фильтров. Результат отображается в строке итогов и в отчётах. Если данные по НДС к вычету отсутствуют, программа покажет предупреждение, но продолжит работу (тогда НДС продажи будет равен начисленному НДС).

📄 Примеры использования
Ежемесячный отчёт для руководства: загрузите данные за месяц, отфильтруйте по нужной компании, нажмите «Экспорт в PDF» — готовый документ можно отправить по электронной почте.

Сравнение двух компаний: загрузите данные по обеим организациям, используйте фильтр для поочерёдного просмотра или сводные графики сразу покажут долю каждой.

Анализ продаж: на вкладке графиков вы увидите ТОП‑5 товаров и распределение прибыли по группам — это поможет в планировании закупок.

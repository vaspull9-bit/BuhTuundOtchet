BuhTuundOtchet
Профессиональный бухгалтерский анализ

Описание
BuhTuundOtchet — это профессиональная настольная программа для бухгалтеров, работающих в системе 1С. Она автоматизирует процесс сбора, анализа и визуализации бухгалтерских данных, позволяя в считанные минуты получать готовые отчёты для руководства и налоговых органов.

Программа разработана специально для российского бухгалтерского учёта, поддерживает работу с несколькими компаниями, детализацию по товарам и товарным группам, а также экспорт результатов в Excel, PDF и Word с профессиональным оформлением.

Ключевые возможности
✓ Импорт данных из Excel-файлов, выгруженных из 1С
✓ Автоматический расчёт: выручка с НДС и без НДС, затраты с НДС и без НДС, валовая прибыль (с НДС), прибыль без НДС, норма прибыли (%), НДС продажи, НДС покупки, НДС в бюджет, налог на прибыль (25%)
✓ Хранение данных во встроенной SQLite-базе
✓ Умная фильтрация по компаниям, периодам, товарным группам
✓ Интерактивная таблица с сортировкой по любому столбцу
✓ 9 профессиональных графиков для анализа: распределение прибыли по товарным группам, ТОП-5 товаров по прибыльности, закупки с НДС по кварталам, выручка с НДС по кварталам, НДС в бюджет по кварталам, НДС по выручке по кварталам, НДС по затратам по кварталам, валовая прибыль по кварталам, затраты по кварталам (все налоги)
✓ Экспорт отчётов в профессиональном формате: Excel (с автоформатированием и итогами), PDF (с титульным листом, таблицами, графиками и анализом), Word (с полноценным структурированным отчётом)
✓ Быстрый отчёт с копированием в буфер обмена
✓ Сохранение настроек путей между сессиями
✓ Современный тёмный интерфейс, удобный для ежедневной работы

Системные требования
Операционная система: Windows 7/10/11 (64-bit)
Процессор: 1.5 GHz или выше
Оперативная память: минимум 2 ГБ (рекомендуется 4 ГБ)
Свободное место: 200 МБ (плюс место для сохраняемых отчётов)
Для запуска из исходного кода: Python 3.8 или выше

Установка

Вариант 1: Готовый исполняемый файл (рекомендуется)

Скачайте последнюю версию BuhTuundOtchet.exe из раздела Releases

Поместите файл в любую папку (например, C:\BuhTuundOtchet)

Запустите двойным щелчком мыши — программа готова к работе

Вариант 2: Запуск из исходного кода

Убедитесь, что установлен Python 3.8+ и pip

Склонируйте репозиторий или скачайте файл buh_tuund.py

Откройте терминал в папке с файлом:
python -m venv venv
venv\Scripts\activate
pip install pyqt6 pandas numpy matplotlib reportlab python-docx openpyxl

Запустите программу: python buh_tuund.py

Подготовка данных в 1С
Для корректной работы программы необходимо выгрузить из 1С следующие отчёты (формат Excel):

ОСВ по счёту 90 (с детализацией по номенклатуре)

ОСВ по счёту 91

ОСВ по счёту 41

Анализ счёта 44

ОСВ по счёту 60

ОСВ по счёту 62

ОСВ по счёту 68

Отчёт по продажам (валовая прибыль)

Книга покупок (за каждый квартал)

Книга продаж (за каждый квартал)

Программа автоматически распознаёт тип отчёта по содержимому и загружает данные в соответствующие поля базы данных.

Работа с программой

Главное меню
База данных — управление базой (загрузка, сохранение, очистка, экспорт)
Отчеты — быстрый отчёт, экспорт в PDF/Word
Настройки — пути к папкам для загрузки и сохранения
Помощь — информация о программе

Панель инструментов
[ОБРАБОТАТЬ] — запускает импорт выбранных файлов
[Экспорт в Excel] — сохраняет данные в Excel
[Экспорт в PDF] — формирует профессиональный PDF-отчёт
[Экспорт в Word] — создаёт структурированный Word-документ
[Быстрый отчёт] — показывает сводку с возможностью копирования
[Настройки] — открывает окно настроек
[О программе] — информация о версии и разработчике

Фильтрация данных
Используйте выпадающие списки для выбора: компании, периода (месяц.год), товарной группы. Нажмите "Применить фильтр" для обновления таблицы и графиков.

Экспорт отчётов
При экспорте в PDF и Word формируется полноценный документ: титульный лист с названием компании и периодом, таблица с финансовыми показателями, все 9 графиков (каждый на отдельной странице), детальные данные, анализ финансового состояния с выводами и рекомендациями.

Финансовые показатели
Выручка с НДС — Σ sales_amount_with_vat
Выручка без НДС — Σ sales_amount_without_vat
Затраты с НДС — Σ purchase_amount_with_vat
Затраты без НДС — Σ (purchase_amount_with_vat - vat_deductible)
Валовая прибыль (с НДС) — Выручка с НДС - Затраты с НДС
Прибыль без НДС — Выручка без НДС - Затраты без НДС
Норма прибыли — (Прибыль без НДС / Выручка без НДС) × 100%
НДС продажи — Σ vat_to_budget
НДС покупки — Σ vat_deductible
НДС в бюджет — НДС продажи - НДС покупки
Налог на прибыль — Прибыль без НДС × 25%

BuhTuundOtchet — это профессиональная настольная программа для бухгалтеров,
работающих в системе 1С. Она автоматизирует процесс сбора, анализа и
визуализации бухгалтерских данных, позволяя в считанные минуты получать
готовые отчёты для руководства и налоговых органов.

Программа разработана специально для российского бухгалтерского учёта,
поддерживает работу с несколькими компаниями, детализацию по товарам и
товарным группам, а также экспорт результатов в Excel, PDF и Word с
профессиональным оформлением.

▸▸▸ ВАРИАНТ 1: ГОТОВЫЙ ИСПОЛНЯЕМЫЙ ФАЙЛ (РЕКОМЕНДУЕТСЯ)

  1. Скачайте последнюю версию BuhTuundOtchet.exe из раздела Releases
  2. Поместите файл в любую папку (например, C:\BuhTuundOtchet)
  3. Запустите двойным щелчком мыши — программа готова к работе

▸▸▸ ВАРИАНТ 2: ЗАПУСК ИЗ ИСХОДНОГО КОДА

  1. Убедитесь, что установлен Python 3.8+ и pip
  2. Склонируйте репозиторий или скачайте файл buh_tuund.py
  3. Откройте терминал в папке с файлом:

     python -m venv venv
     venv\Scripts\activate
     pip install pyqt6 pandas numpy matplotlib reportlab python-docx openpyxl

  4. Запустите программу:
     python buh_tuund.py

  [📝 Экспорт в Word] — создаёт структурированный Word-документ
